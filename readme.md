🍌 Nano Banana API (Gemini Image Bridge)

Неофициальный API-мост, который превращает веб-версию Google Gemini в полноценный сервер генерации изображений. Создан специально для работы в связке с SillyTavern и другими фронтендами, поддерживающими формат OpenAI (DALL-E).
✨ Главные фишки

    Умная кроссплатформенная авторизация: Скрипт сам понимает, где он запущен. На ПК он откроет удобное окно браузера, а на Android (Termux) поможет авторизоваться вручную без ошибок совместимости.

    Поддержка референсов (Image2Image): Вы можете отправлять картинки вместе с промптом, скрипт автоматически загрузит их на серверы Гугла как референсы для генерации.

    Две модели на выбор:

        nano-banana-pro — Генерация в два этапа с улучшением (Redo with Pro). Максимальное качество.

        nano-banana-2 — Базовая генерация (1 этап). Быстрее, отлично подходит для тестов.

💻 Установка и запуск (Для Windows / Mac / Linux)

Для работы потребуется установленный Python 3.

    Скачайте файлы проекта в пустую папку (Нажмите ПКМ -> Открыть в терминале в удобном месте):
	
git clone https://github.com/l4ckofsleep/geminiweb2api.git

cd geminiweb2api

    Откройте терминал/консоль в этой папке и установите зависимости:

pip install Flask Flask-Cors requests playwright

playwright install chromium

    Запустите умный лаунчер:

python start.py

    При первом запуске: Скрипт скачает Chromium и откроет окно Google. Вам нужно будет авторизоваться в свой (желательно фейковый) аккаунт, дождаться прогрузки Gemini и нажать ENTER в консоли. Скрипт сохранит вашу сессию и запустит сервер на порту 1717.

📱 Установка и запуск (Для Android / Termux)

На мобильных устройствах скрипт работает максимально легко, не требуя тяжелых браузерных движков.

    Установите приложение Termux.

    Выполните базовую настройку и установите пакеты:

pkg update && pkg upgrade

pkg install python

pip install Flask Flask-Cors requests
	
	Затем, пулльните сам проект:
	
cd

git clone https://github.com/l4ckofsleep/geminiweb2api.git

cd geminiweb2api

    Запустите лаунчер:

python start.py

    При первом запуске: Скрипт поймет, что вы на Android, и попросит ввести токены вручную.

        Установите из Google Play браузер Kiwi Browser или Firefox.

        Установите в нем расширение Cookie-Editor.

        Зайдите на gemini.google.com и авторизуйтесь.

        Откройте Cookie-Editor и скопируйте значения куков __Secure-1PSID и SAPISID. Вставьте их в консоль Termux по запросу.

🔄 Управление сессиями и Решение проблем

Вечных сессий не бывает. Если вы сменили пароль от Google, нажали кнопку "Выйти" в браузере или Гугл просто обнулил токены (обычно они живут месяцами), генерация перестанет работать, а в консоли полетят ошибки.

Чтобы сбросить старую сессию и авторизоваться заново, просто запустите скрипт с флагом:

python start.py --reauth

Скрипт удалит старый файл google_state.json и предложит пройти авторизацию с чистого листа.
